KaryScript

KaryScript used to be an experimental language for me to test a few different ideas on code readability.

Installation

KaryScript is available via the npm registry. You can install it via:

bash
% npm install -g karyscript


bash
% npm install -g karyscript

And then the compiler can be used with the command kc. To see what the compiler can do you can use the --help flag.

Definitions

You can define variables and constants in the KaryScript with the fix and mut keywords. As you can imagine the mut stands for mutability and the fix for not being assignable again.

karyscript
// KaryScript
fix x = 10
mut y = 20


karyscript
// KaryScript
fix x = 10
mut y = 20
js
// JavaScript
const x = 10
let   y = 20


js
// JavaScript
const x = 10
let   y = 20

Strings

KaryScript accepts both Strings via " and ' quotes. String interpolation is done via the octothorpe # behind a function. So you can put any S-Expression after it.

karyscript
fix x = "hello #(get-user-name pouya)"
fix y = 'hello'


karyscript
fix x = "hello #(get-user-name pouya)"
fix y = 'hello'

Module Imports

// KaryScript
use fs, path

Tables

A huge source of readability issues is the representation of data in a linear format. Writing data structures of multiple dimensions inside of a linear language is hard. KaryScript tries to provide a few new syntax sugars that help you define some of the cases more naturally.

One, is to have the Markdown Tables inside of a programming language. The names in the top will be the object names and then the rows can hold any values. So you can do things as crazy as this:

karyscript
fix phone-book =
    | name      | surname       | age       |
    |-----------|---------------|-----------|
    | "Homer"   | "Simpsons"    | 38        |
    | "Lisa"    | "Simpsons"    | 8         |


karyscript
fix phone-book =
    | name      | surname       | age       |
    |-----------|---------------|-----------|
    | "Homer"   | "Simpsons"    | 38        |
    | "Lisa"    | "Simpsons"    | 8         |

And this is equivalent to the JavaScript:

js
const phoneBook = [
    {
        name:       "Homer",
        surname:    "Simpsons",
        age:        38,
    }, {
        name:       "Lisa",
        surname:    "Simpsons",
        age:        8,
    }
]


js
const phoneBook = [
    {
        name:       "Homer",
        surname:    "Simpsons",
        age:        38,
    }, {
        name:       "Lisa",
        surname:    "Simpsons",

fix phone-book =

    | lisa  | "Lisa"    | "Simpsons"    | 8         |
